# Cognitive Labs Website

This is the static website for Cognitive Labs LLC.

## Overview

The website is a static HTML/CSS/JS site designed to be lightweight and fast.

## Project Structure

- `index.html`: Main landing page.
- `styles.css`: Main stylesheet.
- `main.js`: Main JavaScript file for interactivity.
- `privacy.html`: Privacy Policy page.
- `terms.html`: Terms of Service page.

## deployment

This site can be deployed to any static site hosting provider (e.g., GitHub Pages, Netlify, Vercel, AWS S3).

## Local Development

To run the site locally, you can use a simple HTTP server, for example with Python:

```bash
python3 -m http.server
```

Then open `http://localhost:8000` in your browser.
